# ✦ ZCLOTHING — The Atelier of Style

"Where luxury meets identity — a complete fashion e-commerce experience"

**Live Demo:** https://prashamdamani.github.io/ZClothing-A-FashionStore-/


## About

ZCLOTHING is a fully functional fashion e-commerce platform built entirely with vanilla HTML, CSS, and JavaScript — no frameworks, no libraries. It delivers a complete shopping experience with user authentication, dynamic product filtering, shopping cart, wishlist, professional checkout simulation, order tracking, and personalized dashboard.


## Key Features

**Shopping:** Dynamic product grid with category, price, color, and size filters | Real-time search | Smart sorting | Wishlist | Cart with quantity management

**Authentication:** User registration and login | Session persistence | Logout functionality

**Checkout:** Multi-step form | Form validation | Payment simulation with progress bar | COD, Card, UPI options | Tax calculation (5% GST)

**Dashboard:** Order history | Wishlist view | Lifetime spending analytics | Personalized greeting

**Design:** Premium gold/cream aesthetic | Fully responsive | Smooth animations | Toast notifications | Cart drawer

---

## Tech Stack

- HTML5 — Semantic structure
- CSS3 — Custom properties, Flexbox, Grid, animations
- JavaScript (ES6+) — Vanilla, no dependencies
- localStorage — Persistent data storage
- Google Fonts — Premium typography
